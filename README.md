# syllabary

> 日语50音

### 介绍
包括三个页面：
1. 50音图页
    * 平假名 / 片假名 切换
    * 开始评测
    * 生词表
2. 随机评测页
    * 平假名 / 片假名 切换
    * 返回50音
    * 进入生词表
3. 生词表页
    * 生词表列表
    * 清空生词表
    * 开始评测
